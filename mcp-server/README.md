# mcp-server

**Required.** MCP tool server (Streamable HTTP).

- URL: `http://127.0.0.1:8788/mcp` (see `MCP_PORT`)
- Run: `pnpm dev:mcp` from repo root
- Add tools in `tools/` — start from `tools/sample-employees.ts`
